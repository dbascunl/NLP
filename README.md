# NLP - Procesamiento de lenguaje natural

Repositorio con una colección estructurada de experimentos y notebooks enfocados a tareas de clasificación, embeddings, topic modeling, modelos basados en transformers encoderes y decoderes, así como técnicas modernas (RAG). Lo anterior incluye:

* Preprocesamiento avanzado de texto
* Pipelines reproducibles con uso de liberías de interés como (HuggingFace, Pytorch, TensorFlow, Sklearn, entre otras).
* Análisis Comparativos de modelos de embeddings junto con aplicaciones hacía modelado de tópicos mediante SentenceTransformes (visualización Semántica)
* Implementación de piplines de RAG (Retrieval-Augmented Generation)
* Integración de LLMs mediante LangChain y DSPy para resolución de generación de texto con RAG.
